# 📘 Assignment 30 – File Handling in Python

## 🧾 Assignment Title 

**Advanced File Processing using Python**

--- 

## 🎯 Aim

The aim of this assignment is to understand and implement different **file processing techniques** in Python such as:

* Counting lines and words in a file
* Reading file contents line by line
* Copying contents from one file to another
* Searching a specific word inside a file

These programs help in building strong fundamentals of **file handling and text processing**.

## 📂 Assignment Problems

---

### 🔹 Q1) Count Lines in a File

**Problem Statement:**
Write a program which accepts a file name from the user and counts how many lines are present in the file.

**Input:**
Demo.txt

**Expected Output:**
Total number of lines in `Demo.txt`.

**Concepts Used:**

* File reading
* Line counting logic

---

### 🔹 Q2) Count Words in a File

**Problem Statement:**
Write a program which accepts a file name from the user and counts the total number of words in that file.

**Input:**
Demo.txt

**Expected Output:**
Total number of words in `Demo.txt`.

**Concepts Used:**

* File reading
* Word separation
* String processing

---

### 🔹 Q3) Display File Line by Line

**Problem Statement:**
Write a program which accepts a file name from the user and displays the contents of the file line by line on the screen.

**Input:**
Demo.txt

**Expected Output:**
Display each line of `Demo.txt` one by one.

**Concepts Used:**

* File iteration
* Line-by-line reading

---

### 🔹 Q4) Copy File Contents into Another File

**Problem Statement:**
Write a program which accepts two file names from the user.

* First file is an existing file
* Second file is a new file

Copy all contents from the first file into the second file.

**Input:**
ABC.txt Demo.txt

**Expected Output:**
Contents of `ABC.txt` copied into `Demo.txt`.

**Concepts Used:**

* File read and write operations
* File copy logic

---

### 🔹 Q5) Search a Word in File

**Problem Statement:**
Write a program which accepts a file name and a word from the user and checks whether that word is present in the file or not.

**Input:**
Demo.txt Marvellous

**Expected Output:**
Display whether the word `Marvellous` is found in `Demo.txt` or not.

**Concepts Used:**

* File reading
* String searching

---

## 📌 Learning Outcomes

After completing this assignment, you will be able to:

* Count lines and words in text files
* Process files line by line
* Copy data between files safely
* Search specific words within a file
