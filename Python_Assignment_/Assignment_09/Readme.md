## Questions Included
Question 1

Write a program which contains one function named as Display() that prints
"Jay Ganesh" on console.

Question 2

Write a program which contains one function ChkGreater() that accepts two numbers
and prints the greater number.

Input: 10 20
Output: 20 is greater

Question 3

Write a program which accepts one number and prints the square of that number.

Input: 5
Output: 25

Question 4

Write a program which accepts one number and prints the cube of that number.

Question 5

Write a program which accepts one number and checks whether it is divisible by 3 and 5.

## How to Run the Programs

Use the following command:

python program09_X.py


Replace X with the question number.
